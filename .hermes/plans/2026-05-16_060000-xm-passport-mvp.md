# XM Passport — Implementation Plan

> **For Hermes:** Execute sequentially. Each section builds on prior.

**Goal:** Mobile-first collector ecosystem app — scan metal cards, build digital vault, earn points, unlock ranks, trade verified e-tickets, redeem for physical statues.

**Architecture:** Next.js 14 App Router (React Server Components + API routes), SQLite via better-sqlite3, Tailwind CSS mobile-first, PWA with service worker. Single VPS deploy on port 3001.

**Tech Stack:** Next.js 14, React 18, Tailwind CSS 3, better-sqlite3, html5-qrcode, next-pwa, bcryptjs, jsonwebtoken, uuid

## Database Schema

### users
```sql
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  email TEXT UNIQUE NOT NULL,
  password_hash TEXT NOT NULL,
  display_name TEXT NOT NULL,
  collector_number TEXT,        -- e.g. "XM-00042"
  avatar_url TEXT,
  total_points INTEGER DEFAULT 0,
  rank TEXT DEFAULT 'bronze',   -- bronze, silver, gold, platinum, master
  rank_points INTEGER DEFAULT 0, -- points in current rank tier
  collection_count INTEGER DEFAULT 0,
  verified_collector BOOLEAN DEFAULT 0,
  created_at TEXT DEFAULT (datetime('now')),
  updated_at TEXT DEFAULT (datetime('now'))
);
```

### cards
```sql
CREATE TABLE cards (
  id TEXT PRIMARY KEY,           -- UUID
  card_code TEXT UNIQUE NOT NULL, -- QR code payload "XM-CARD-XXXX"
  product_name TEXT NOT NULL,     -- "Batman Who Laughs Metal Card"
  ip TEXT,                        -- IP franchise
  edition TEXT,                   -- "Limited Edition"
  edition_size INTEGER,           -- 500
  edition_number INTEGER,         -- 042/500
  rarity TEXT DEFAULT 'common',   -- common, rare, ultra-rare, legendary
  card_type TEXT DEFAULT 'metal', -- metal, holo, premium
  image_url TEXT,
  owner_id TEXT REFERENCES users(id),
  status TEXT DEFAULT 'active',   -- active, traded, locked
  scanned_at TEXT,
  points_value INTEGER DEFAULT 100,
  created_at TEXT DEFAULT (datetime('now'))
);
```

### etickets
```sql
CREATE TABLE etickets (
  id TEXT PRIMARY KEY,
  ticket_code TEXT UNIQUE NOT NULL, -- "XM-ET-XXXX"
  product_id TEXT NOT NULL,         -- linked statue product
  product_name TEXT NOT NULL,
  owner_id TEXT REFERENCES users(id),
  status TEXT DEFAULT 'active',     -- active, redeemed, traded, expired
  payment_status TEXT DEFAULT 'reserved', -- reserved, paid, refunded
  purchase_price REAL,
  redemption_date TEXT,
  expiry_date TEXT,
  points_earned INTEGER DEFAULT 0,
  created_at TEXT DEFAULT (datetime('now'))
);
```

### products
```sql
CREATE TABLE products (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  ip TEXT,
  description TEXT,
  product_type TEXT DEFAULT 'statue', -- statue, card_set, exclusive
  edition_size INTEGER,
  price REAL,
  image_url TEXT,
  release_date TEXT,
  status TEXT DEFAULT 'draft',        -- draft, announced, preorder, soldout, shipped
  e_ticket_enabled BOOLEAN DEFAULT 1,
  created_at TEXT DEFAULT (datetime('now'))
);
```

### marketplace_listings
```sql
CREATE TABLE marketplace_listings (
  id TEXT PRIMARY KEY,
  seller_id TEXT REFERENCES users(id),
  item_type TEXT NOT NULL,         -- card, eticket, collectible
  item_id TEXT NOT NULL,
  price REAL NOT NULL,
  currency TEXT DEFAULT 'SGD',
  status TEXT DEFAULT 'active',    -- active, sold, cancelled
  buyer_id TEXT REFERENCES users(id),
  sold_at TEXT,
  created_at TEXT DEFAULT (datetime('now'))
);
```

### points_transactions
```sql
CREATE TABLE points_transactions (
  id TEXT PRIMARY KEY,
  user_id TEXT REFERENCES users(id),
  amount INTEGER NOT NULL,
  reason TEXT NOT NULL,             -- card_scan, eticket_purchase, set_complete, etc.
  reference_id TEXT,                -- related card/ticket ID
  created_at TEXT DEFAULT (datetime('now'))
);
```

### collection_journeys
```sql
CREATE TABLE collection_journeys (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,               -- "Batman Rogues Gallery"
  description TEXT,
  ip TEXT,
  required_items TEXT NOT NULL,     -- JSON array of product/card IDs
  reward_points INTEGER DEFAULT 500,
  reward_rank_boost INTEGER DEFAULT 0,
  badge_url TEXT,
  created_at TEXT DEFAULT (datetime('now'))
);
```

### user_journeys
```sql
CREATE TABLE user_journeys (
  id TEXT PRIMARY KEY,
  user_id TEXT REFERENCES users(id),
  journey_id TEXT REFERENCES collection_journeys(id),
  progress TEXT DEFAULT '{}',       -- JSON {item_id: collected}
  completed BOOLEAN DEFAULT 0,
  completed_at TEXT,
  created_at TEXT DEFAULT (datetime('now'))
);
```
